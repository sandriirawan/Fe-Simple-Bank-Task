### Installation

- Clone This Front End Repo

```
git clone https://github.com/sandriirawan/Fe-Simple-Bank-Task
```

- Go To Folder Repo

```
cd Fe-Simple-Bank-Task
```

- Install Module

```
yarn || npm install
```
